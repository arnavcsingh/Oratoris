# Oratoris

Oratoris is a **modern, AI-powered public speaking assistant** that helps users improve their speech performance by providing **real-time transcription, filler word analysis, and speaking speed visualization**. Designed with a **space-inspired aesthetic**, Oratoris delivers a polished, professional, and engaging user experience.

---

## Features

- **Audio Upload & Recording**
  - Upload pre-recorded speech or record directly within the app.
  - Supports standard audio formats (`.webm`, `.mp3`, `.wav`, etc.).

- **Transcription**
  - Automatic transcription of your speech in real-time.
  - Clear display of spoken content for easy review.

- **Filler Word Detection**
  - Highlights filler words such as “um,” “uh,” and “like.”
  - Provides a count for each filler word used.

- **Speaking Speed (WPM) Visualization**
  - Interactive line chart showing words per minute (WPM) over time.
  - Helps identify pacing and maintain consistent speech rhythm.

- **Modern, Space-Themed UI**
  - Elegant dark mode with floating stars and animated background elements.
  - 3D-inspired cards and interactive buttons.
  - Designed to be **attention-grabbing and immersive**.

---

## Technologies

- **Frontend:** React 19, Tailwind CSS 4  
- **Charts:** Recharts  
- **Audio Handling:** MediaRecorder API, HTML5 `<audio>`  
- **Animations & Effects:** react-tsparticles, CSS transforms & shadows
